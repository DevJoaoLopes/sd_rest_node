# PO Sistemas Distribuidos

## Criar um WebService (API) REST
- Lista alunos
- Pesquisa um aluno e traz dados desse aluno
- Edita um aluno
- Remove um aluno


## Entidade aluno
- Nome
- Curso
- Semestre
- RA
- CPF
- Cidade


## Tecnologia usada
- Node JS + Express
